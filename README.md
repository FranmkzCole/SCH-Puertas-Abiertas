# 🏫 SCH - Puertas Abiertas

**Schonthal Puertas Abiertas** es una página web informativa del Colegio Schonthal, diseñada para mantener a la comunidad educativa al día con noticias, eventos y actividades institucionales.

## 🌐 Descripción del Proyecto

Este proyecto tiene como objetivo funcionar como un medio digital similar a un diario escolar, donde los estudiantes podrán acceder fácilmente a información relevante sobre el colegio.

La plataforma publicará novedades, proyectos, eventos especiales y comunicados importantes, facilitando que los alumnos estén informados de manera clara, accesible y actualizada.

## 🎯 Objetivos

- Centralizar la información institucional del colegio  
- Mantener informados a los alumnos sobre eventos y actividades  
- Crear un espacio digital dinámico y accesible  
- Fomentar la participación y el interés de la comunidad educativa  

## 🛠️ Tecnologías Utilizadas

- **Desarrollo Web** (HTML, CSS, JavaScript)  
- **Diseño UI/UX** enfocado en accesibilidad y claridad  
- **Gestión de contenido** para publicaciones tipo blog/diario
